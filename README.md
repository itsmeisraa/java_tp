# Java Learning Series - TPs

This repository is dedicated to a structured learning journey through Java. It contains various practical exercises (TPs) covering fundamental to advanced programming concepts.

## 🚀 Learning Roadmap

- **Module 1: Basics**
  - Syntax, Variables, and Data Types
  - Control Flow (If-Else, Loops)
  - Method creation and Scope

- **Module 2: Object-Oriented Programming (OOP)**
  - Encapsulation (Getters/Setters)
  - Inheritance & Polymorphism
  - Interfaces and Abstract Classes

- **Module 3: Collections & Data**
  - Working with `ArrayList` and `HashMap`
  - Handling Exceptions (`try-catch`)
  - File I/O operations
